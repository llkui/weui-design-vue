```html

  <div>
```

## Attributes

| 参数 | 说明           | 类型    | 可选值                                                       | 默认值  |
| ---- | -------------- | ------- | ------------------------------------------------------------ | ------- |
| type | 类型           | string  | success / info / warn-primary / warn / waiting / info-circle | success |
| mini | 是否为mini状态 | boolean | —                                                            | false   |
