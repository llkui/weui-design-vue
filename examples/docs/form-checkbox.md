## Attributes

### checkbox-group

| 参数           | 说明         | 类型    | 可选值 | 默认值 |
| -------------- | ------------ | ------- | ------ | ------ |
| name           | 标识符       | string  | —      | —      |
| value(v-model) | 当前选中的值 | array   | —      | —      |
| showMore       | 是否显示更多 | boolean | —      | false  |

| 事件名称 | 说明         | 回调参数   |
| -------- | ------------ | ---------- |
| onMore   | 更多事件回调 | function() |

### checkbox

| 参数  | 说明     | 类型   | 可选值 | 默认值 |
| ----- | -------- | ------ | ------ | ------ |
| value | 选项的值 | string | —      | —      |
| label | 选项名   | string | —      | —      |
